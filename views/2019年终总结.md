
### 前言

和SQL数据库一样，MongoDB也存在注入式攻击的风险

##### 操作符注入


比较条件操作符
```
"$lt", "$lte", "$gt", "$gte", "$ne"
对应于："<","<=",">",">=","!=" 
```

正则操作符

```
$regex
```

运用
 
 利用操作符，实现对mongo的注入

例子：

```
db.items.find({"name" :"admin", "password" : {$ne : "1"}})
#查询name=admin及password不等以1的数据
相当于SQL语句：select * from item where name='admin' and password != 1;
db.items.find({"name" :{'$regex':"^a"}})
#查询正则匹配字母a开头的数据

```
